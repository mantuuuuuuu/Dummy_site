HTML5 markup

HTML5 (Hypertext Markup Language 5) is the latest version of HTML, the markup language used to structure and present content on the World Wide Web. HTML5 was developed by the World Wide Web Consortium (W3C) and was released in 2014.

CSS3 Styling

CSS3 (Cascading Style Sheets 3) is the latest version of the CSS language used to style and format web pages. It was first released in 1999, and the latest version, CSS3, was released in 2011.

Graphic Design

Graphic design is the art and practice of creating visual content to communicate information and ideas to a specific audience. It involves the use of various elements such as typography, imagery, color, and layout to create a visual representation of a message or concept.
